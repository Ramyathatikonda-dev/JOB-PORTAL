# 💼 Smart Job Portal

## 🔹 Project Description
A full-stack job portal web application developed using Django. Users can register, login, search jobs, and apply. Admin can manage job postings and applications.

## 🔹 Features
- User Registration & Login
- Job Listings & Search
- Apply for Jobs
- Admin Dashboard
- Responsive Design

## 🔹 Technologies Used
- Python (Django)
- HTML, CSS, JavaScript
- Bootstrap
- SQLite

## 🔹 How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python manage.py runserver# JOB-PORTAL
